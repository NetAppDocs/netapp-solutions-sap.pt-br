= Instalação do software SAP HANA

[role="lead"]
Veja a seguir os requisitos de instalação de software para SAP HANA.



== Instalar em sistema de host único

A instalação do software SAP HANA não requer preparação adicional para um sistema de host único.



== Instale em sistema de vários hosts

Para instalar o SAP HANA em um sistema de vários hosts, execute as seguintes etapas:

. Usando a ferramenta de instalação SAP `hdbclm`, inicie a instalação executando o seguinte comando em um dos hosts de trabalho. Use a `addhosts` opção para adicionar o segundo trabalhador (`sapcc-hana-tst-03`) e o host de (`sapcc-hana-tst-04`reserva ).
+
....
apcc-hana-tst-02:/mnt/sapcc-share/software/SAP/HANA2SPS7-73/DATA_UNITS/HDB_LCM_LINUX_X86_64 #
./hdblcm --action=install --addhosts=sapcc-hana-tst-03:role=worker,sapcc-hana-tst-04:role=standby
