= Implante o SAP HANA no SUSE KVM com armazenamento NetApp usando SR-IOV e NFS.
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
Implante o SAP HANA Single-Host no SUSE KVM usando armazenamento NetApp com interfaces de rede SR-IOV e acesso ao armazenamento via NFS ou FCP.  Siga este fluxo de trabalho para configurar interfaces virtuais, atribuí-las a máquinas virtuais e configurar conexões de armazenamento para obter o desempenho ideal.

Para obter uma visão geral do SAP HANA em virtualização KVM, consulte a documentação da SUSE: https://documentation.suse.com/sbp/sap-15/pdf/SBP-SLES4SAP-HANAonKVM-SLES15SP5_en.pdf["Melhores práticas da SUSE para SAP HANA em KVM"] .

image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["Um"]link:kvm-hana-requirements.html["Revise os requisitos de configuração"]

[role="quick-margin-para"]
Analise os principais requisitos para a implementação do SAP HANA no SUSE KVM usando armazenamento NetApp com SR-IOV e protocolos de armazenamento.

image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["Dois"]link:kvm-hana-network.html["Configurar interfaces de rede SR-IOV"]

[role="quick-margin-para"]
Configure o SR-IOV (Virtualização de E/S de Raiz Única) no host KVM e atribua interfaces virtuais à máquina virtual para comunicação de rede e acesso ao armazenamento.

image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["Três"]link:kvm-hana-fcn.html["Configurar rede Fibre Channel"]

[role="quick-margin-para"]
Atribua portas HBA FCP físicas à VM como dispositivos PCI para usar LUNs FCP com o SAP HANA.

image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png["Quatro"]link:kvm-hana-storage.html["Configure o armazenamento NetApp para SAP HANA."]

[role="quick-margin-para"]
Configure conexões de armazenamento NFS ou FCP entre a máquina virtual e os sistemas de armazenamento NetApp para os arquivos do banco de dados SAP HANA.
