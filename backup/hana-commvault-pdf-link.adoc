= Backup e recuperação do SAP HANA usando NetApp Storage Systems e o software Commvault
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
Este documento descreve o projeto de uma solução da NetApp e Commvault para SAP HANA, que inclui a tecnologia de gerenciamento de snapshots Commvault IntelliSnap e a tecnologia Snapshot da NetApp. A solução é baseada no armazenamento NetApp e no pacote de proteção de dados Commvault.

Autores: Marco Schoen, NetApp; Dr. Tristan Daude, CommVault Systems
