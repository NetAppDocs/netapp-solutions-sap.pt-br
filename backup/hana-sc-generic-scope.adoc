= Proteja sistemas SAP HANA com o SnapCenter em ONTAP, Azure NetApp Files e FSx para ONTAP.
:allow-uri-read: 


[role="lead"]
Proteja sistemas SAP HANA com o NetApp SnapCenter usando backups baseados em snapshots e replicação de dados. Esta solução abrange as melhores práticas de configuração e operação do SnapCenter para sistemas SAP HANA em sistemas ONTAP AFF e ASA , Azure NetApp Files e Amazon FSx para ONTAP, incluindo estratégias de backup, verificações de consistência e fluxos de trabalho de recuperação.

Autor: Nils Bauer, NetApp
